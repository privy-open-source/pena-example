# Pena (Vanilla) + Vite (TS)

## Prerequisites

- [Node](https://nodejs.org/en/download) 18+
- Yarn 2 *(Optional)*

## Running Local

1. Install deps

```bash
npm install
```

2. Open `src/main.ts` and replace `YOUR_DOCUMENT_URL` with your actual document'url
3. Run dev server

```bash
npm run dev
```
Then, open `http://localhost:5173` with your browser
