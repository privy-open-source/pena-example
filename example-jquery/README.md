# Pena (jQuery) + Vite

## Prerequisites

- [Node](https://nodejs.org/en/download) 18+
- Yarn 2 *(Optional)*

## Running Local

1. Install deps

```bash
npm install
```

1. Open `index.html` and replace `YOUR_DOCUMENT_URL` with your actual document'url
2. Run dev server

```bash
npm run dev
```
Then, open `http://localhost:5173` with your browser
